# A BTC Alert App

### Getting started

clone this repository

## Install dependencies

```
  npm i | yarn
```

:warning: Add a `.env` file on your project root and add this :warning:

```
NODE_PATH=./src
REACT_APP_XCOIN_API_KEY=<your_xcoin_api_key>
```

## Start the dev server
```
npm start | yarn start
```
